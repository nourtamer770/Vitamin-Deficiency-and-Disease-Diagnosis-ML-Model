# Vitamin-Deficiency-ML-Model
Vitamin Deficiency & Disease Diagnosis Prediction System

This project implements a complete end-to-end machine learning pipeline for predicting both vitamin deficiency levels (regression) and disease diagnosis categories (classification) using demographic, lifestyle, and nutritional information.

Key Features
Exploratory Data Analysis (EDA)
Missing value handling and data preprocessing
Feature engineering using nutritional indicators
One-Hot Encoding for categorical variables
Correlation-based and Random Forest feature selection
Regression models:
Linear Regression
Random Forest Regressor
CatBoost Regressor
Classification models:
Logistic Regression
Random Forest Classifier
K-Nearest Neighbors
Support Vector Machine (SVM)
Hyperparameter tuning and model comparison
Performance evaluation using RMSE, R², and Accuracy
Deployment-ready preprocessing pipelines using saved encoders, scalers, and trained models
Technologies Used
Python
Pandas
NumPy
Scikit-learn
CatBoost
Matplotlib
Seaborn
Pickle
Results

The project evaluates multiple machine learning algorithms and feature selection strategies to identify the most effective models for predicting vitamin deficiency severity and disease diagnosis from patient health and nutritional data.
